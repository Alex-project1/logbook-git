# RouteMaster Departments Stage 2

Этот пакет продолжает Stage 1 и обновляет админку под новую логику подразделений.

## Что добавлено в Stage 2

- Сотрудники: выбор города и подразделения, фильтр по подразделению, комментарий/должность.
- Автомобили: выбор города и подразделения, фильтр по подразделению, начальный пробег, комментарий.
- Наряды ГБР: выбор города, только подразделения типа ГШР, логин/пароль при создании, новый пароль при редактировании, архив вместо удаления.
- Доп. посты: выбор города и подразделения, логин/пароль при создании, новый пароль при редактировании, архив вместо удаления.
- API админки обновлены с обратной совместимостью старых вызовов.

## Важное по паролям

Пароль не показывается в редактировании, потому что backend хранит только passwordHash.
Если пароль забыт — задайте новый пароль в полях "Новый пароль" и "Подтвердить новый пароль".

## Установка

Распаковать архив в корень проекта с заменой файлов.

Backend:

```powershell
cd backend
npm install
npx prisma generate
npx prisma db push
node -r dotenv/config seed-system-foundation.js
npm run dev
```

Admin:

```powershell
cd admin
npm install
npm run build
npm run dev
```

Проверка сборки админки уже выполнена: `npm run build` успешно проходит.
